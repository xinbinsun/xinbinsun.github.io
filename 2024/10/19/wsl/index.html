适用于 Linux 的 Windows 子系统 (WSL) 可让开发人员直接在 Windows 上按原样运行 GNU/Linux 环境（包括大多数命令行工具、实用工具和应用程序），且不会产生传统虚拟机或双启动设置开销。

Linux常用命令

1. 文件和目录操作

  • ls：列出目录内容

  • cd：改变当前工作目录

  • pwd：打印当前工作目录

  • mkdir：创建新目录

  • rmdir：删除空目录

  • rm：删除文件或目录

  • cp：复制文件或目录

  • mv：移动或重命名文件/目录

  • touch：创建空文件或更新文件时间戳

2. 文件查看

  • cat：查看文件内容

  • less：分页查看文件内容

  • head：查看文件的开头几行

  • tail：查看文件的结尾几行

  • grep：搜索文件中的文本

3. 系统信息和管理

  • top 或 htop：查看系统运行的进程和资源使用情况

  • ps：显示当前运行的进程

  • kill：终止进程

  • df：显示磁盘空间使用情况

  • du：查看文件或目录的磁盘使用量

4. 文件权限

  • chmod：修改文件权限

  • chown：更改文件所有者

5. 网络操作

  • ping：测试与某个地址的网络连接

  • wget：下载文件

  • curl：从 URL 获取数据

6. 压缩和解压缩

  • tar：打包与解压 tar 文件

  • zip / unzip：压缩和解压 zip 文件

7. 权限提升

  • sudo：以超级用户权限执行命令

8. 搜索和查找

  • find：查找文件或目录

  • locate：快速查找文件

9. 系统更新

  • apt：用于 Ubuntu 等基于 Debian 的发行版

10. 移动windows上的项目至WSL
